# About 

This is a simple python code that extracts detail from ip address

# Module Required

- urllib
- json 

# Installation & Working 

1. Clone the repository
> git clone https://github.com/kmaaz-dev/Ip2Info.git

2. Change the current directory to Ip2Info
> cd Ip2Info

3. Run the script 
>  python3 ip2info.py
